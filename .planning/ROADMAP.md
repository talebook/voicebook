## 2. 阶段详情

### Phase 1: 文本处理与角色分析

**目标**: 建立完整的文本处理流水线，实现角色识别和画像提取

**需求映射**:
- REQ-001 TXT解析
- REQ-002 角色识别
- REQ-003 对话分类
- REQ-004 画像提取
- REQ-008 历史追踪

**成功标准**:
1. ✅ 能解析任意UTF-8编码的小说TXT文件
2. ✅ LLM角色识别准确率≥80%
3. ✅ 对话/旁白分类准确率≥85%
4. ✅ 跨章节角色状态追踪正常

**交付物**:
- `src/text_parser.py` - TXT解析模块
- `src/character_analyzer.py` - 角色分析模块
- `src/dialogue_extractor.py` - 对话提取模块
- `src/models.py` - 数据模型定义
- `tests/test_text_processing.py` - 单元测试

**Plan**:

```
Plan 1.1: TXT解析模块
├── 1.1.1 创建章节分割正则规则
├── 1.1.2 实现章节标题识别
├── 1.1.3 实现段落提取和分类
└── 1.1.4 编写单元测试

Plan 1.2: 对话提取模块
├── 1.2.1 设计对话匹配规则
├── 1.2.2 实现多格式对话识别（""、「」等）
├── 1.2.3 实现说话者归属逻辑
└── 1.2.4 集成LLM进行二次验证

Plan 1.3: 角色分析模块（扩展已有代码）
├── 1.3.1 集成Qwen3-0.6B进行角色识别
├── 1.3.2 实现角色画像提取
├── 1.3.3 实现角色历史追踪
└── 1.3.4 优化Prompt提升准确率
```

### Phase 2: TTS语音合成

**目标**: 集成TTS引擎，实现文本到语音的转换

**需求映射**:
- REQ-005 TTS合成
- REQ-006 音频拼接
- REQ-007 年龄-音色映射

**成功标准**:
1. ✅ 支持Edge TTS和CosyVoice双引擎
2. ✅ 音色映射功能正常
3. ✅ 角色年龄变化时音色切换正确
4. ✅ 单章节TTS合成时间<5分钟

**交付物**:
- `src/tts/edge_engine.py` - Edge TTS引擎
- `src/tts/cosyvoice_engine.py` - CosyVoice引擎
- `src/voice_selector.py` - 音色选择器
- `tests/test_tts.py` - TTS测试

**Plan**:

```
Plan 2.1: TTS引擎集成
├── 2.1.1 实现Edge TTS引擎
├── 2.1.2 实现CosyVoice引擎
├── 2.1.3 设计TTS抽象接口
└── 2.1.4 编写测试验证

Plan 2.2: 音色映射系统
├── 2.2.1 设计年龄-音色映射表
├── 2.2.2 实现角色音色自动选择
├── 2.2.3 支持用户自定义音色映射
└── 2.2.4 测试音色切换效果
```

### Phase 3: 音频拼接与输出

**目标**: 实现音频拼接和多种格式输出

**需求映射**:
- REQ-010 多格式输出
- REQ-011 分章节输出

**成功标准**:
1. ✅ 支持MP3/M4A/WAV格式输出
2. ✅ 音频拼接无明显断层
3. ✅ 可按章节独立生成
4. ✅ 提供CLI和API两种使用方式

**交付物**:
- `src/audio_assembler.py` - 音频拼接模块
- `src/output_formatter.py` - 格式输出模块
- `src/cli.py` - 命令行工具
- `tests/test_audio.py` - 音频测试
- `main.py` - FastAPI服务（扩展）

**Plan**:

```
Plan 3.1: 音频拼接模块

├── 3.1.2 添加淡入淡出效果
```
