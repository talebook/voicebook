# 说话人识别测试语料

从 talebook 测试书库（`/Users/bytedance/github/talebook/tests/library`）自动提取的
**对白+角色密集**选段，覆盖多种文体。由 `build_samples.py` 生成（滑窗扫描：引文数 + 说话动词
多样性打分，每本取最优窗口 ≤3500 字，自动剥离内部章节标题并加合成章节头）。

```bash
uv run python tests/samples/build_samples.py   # 重新提取样本
uv run python tests/samples/run_reports.py     # 批量生成识别报告 → reports/
```

## 剩余已知边界（按优先级）

1. **第一人称"我"**：叙事者对白目前无法归属（芳华）
2. **译文腔归属线索弱**：动作描写代替说话动词（罪与罚/百年孤独 的 unknown 偏多）
3. **画像性别对译名人物偏弱**（奥雷连诺被判 female）
4. ~~长译名~~ ✅ 已支持（·分量、4字音译、昵称别名 阿X/小X/X儿 归一）
