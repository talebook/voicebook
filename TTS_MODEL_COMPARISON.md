# CosyVoice vs kokoro-onnx 对比分析报告

## 概述

本报告对比分析两个 TTS（文本转语音）模型：**CosyVoice-300M-SFT** 和 **kokoro-onnx**，评估它们在**性能**、**效果**和**适用场景**方面的表现。

---

## 一、模型基本信息

### 1.1 CosyVoice-300M-SFT

| 属性 | 值 |
|------|-----|
| **模型名称** | CosyVoice-300M-SFT |
| **开发方** | 阿里巴巴达摩院 |
| **模型大小** | 300M 参数 |
| **类型** | 端到端语音合成模型 |
| **主要语言** | 中文（重点优化） |
| **架构** | 基于 FunAudioLLM |
| **特点** | - 支持零样本音色克隆<br>- 支持多种预训练音色<br>- 情感控制能力<br>- 自然流畅的中文合成 |

### 1.2 kokoro-onnx

| 属性 | 值 |
|------|-----|
| **模型名称** | kokoro-onnx v1.0 |
| **开发方** | 开源社区 |
| **模型大小** | 轻量级 (ONNX 格式) |
| **类型** | 多语言 TTS 模型 |
| **主要语言** | 多语言支持 |
| **架构** | 基于 ONNX 运行时 |
| **特点** | - 极快的加载速度<br>- 低内存占用<br>- 跨平台支持<br>- 多音色库 |

## 二、性能对比

### 2.1 资源占用对比

| 指标 | CosyVoice | kokoro-onnx |
|------|-----------|-------------|
| **模型文件大小** | ~1.5GB+ | ~100MB |
| **加载时间** | 较慢（秒级） | 极快（毫秒级） |
| **运行内存占用** | 高（>1GB） | 低（<200MB） |
| **磁盘空间** | 大 | 小 |

### 2.2 生成速度对比

基于典型场景的预估：

| 文本长度 | CosyVoice 生成时间 | kokoro-onnx 生成时间 |
|----------|-------------------|---------------------|
| 短句（20字） | ~1-2秒 | ~0.2-0.5秒 |
| 中句（100字） | ~3-5秒 | ~1-2秒 |
| 段落（500字） | ~10-15秒 | ~3-5秒 |

**实时因子（RTF）**：
- CosyVoice: ~0.5-0.8x (比真实语音慢)
- kokoro-onnx: ~2-3x (比真实语音快)

## 四、适用场景分析

### 4.1 CosyVoice 适用场景

✅ **推荐使用场景**：
1. **中文有声书** - 自然流畅的中文发音
2. **高质量播客** - 需要优秀的音色和韵律
3. **专业内容制作** - 追求音质和自然度
4. **情感丰富的文本** - 需要表达不同情绪

❌ **不推荐场景**：
1. 资源受限的边缘设备
2. 需要极速响应的实时应用
3. 批量处理超大量文本

### 4.2 kokoro-onnx 适用场景

✅ **推荐使用场景**：
1. **轻量级应用** - 资源受限的环境
2. **实时语音** - 需要快速响应
3. **多语言需求** - 同时需要多种语言
4. **嵌入式设备** - 低内存占用

❌ **不推荐场景**：
1. 高质量中文有声书
2. 对音色要求极高的应用
3. 需要精确情感表达的场景

## 六、技术实现建议

### 6.1 CosyVoice 集成要点

```python
from cosyvoice.cli.cosyvoice import CosyVoice
import torchaudio

# 加载模型
model_dir = 'pretrained_models/CosyVoice-300M-SFT'
cosyvoice = CosyVoice(model_dir)

# 查看可用音色
print("可用音色:", cosyvoice.list_available_spks())

# 生成语音
result = cosyvoice.inference_sft(
    text="你好，这是一段测试语音",
    spk_id="中文女",
    stream=False
)

# 保存音频
torchaudio.save(
    "output.wav",
    result['tts_speech'],
    cosyvoice.sample_rate
)
```

### 6.2 kokoro-onnx 集成要点

```python
from kokoro_onnx import Kokoro
import soundfile as sf

# 加载模型
kokoro = Kokoro(
    "models/kokoro-v1.0.onnx",
    "models/voices-v1.0.bin"
)

# 生成语音
samples, sample_rate = kokoro.create(
    text="Hello, this is a test",
    voice="af_sarah",  # 选择音色
    speed=1.0
)

# 保存音频
sf.write("output.wav", samples, sample_rate)
```
