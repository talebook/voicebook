# book2audio 项目指南

## 项目概述

book2audio 是一个智能有声书工具，将小说TXT转化为多角色有声书。

## 技术栈

- **LLM**: Qwen3-0.6B (Ollama) - 内存≤1GB
- **TTS**: CosyVoice / Edge TTS
- **框架**: FastAPI

## 执行流程

1. Phase 1: 文本处理与角色分析
2. Phase 2: TTS语音合成
3. Phase 3: 音频拼接与输出
