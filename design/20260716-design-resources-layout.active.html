02 · 已确认决策

文档与资源分层。 只有带生命周期后缀的内部 HTML 放在 design/ 根目录；design/resources/ 是唯一允许的子目录例外。
按功能隔离资源。 本次使用 design/resources/qwen3ttsai/，后续功能使用各自子目录，避免文件名冲突和误删共享材料。
允许本地依赖。 HTML 可通过相对路径引用截图、音频、视频、PDF、JSON 等资源；仍禁止远程脚本、样式、字体和渲染时网络依赖。
保留复现能力。 JSON、脚本、音频和 MP4 不内嵌进 Git 历史中的第二份副本；报告直接引用资源，生成器校验 WAV 的 SHA-256。
