TL;DR
Voicebook 已接入 qwen3ttsai.com 的公开系统音色接口，并从 27 个中文音色中按角色性别、年龄阶段和声音描述自动选声。五个小说片段全部生成成功；并发 2 时用 12.307 秒生成 47.445 秒音频，有效 RTF 为 0.259。WAV 与 MP4 统一放在 design/resources/qwen3ttsai/，本报告可随该目录离线播放。

01API 协议与接入链路

ENDPOINT

RESILIENCE

并发 2 · 最多 3 次

SERVICE BOUNDARY

05运行与验证

生成 Qwen 有声书

uv run python -m book2audio \
  --input book/xuanjian.txt --chapters 1 \
  --output output/xuanjian_qwen.mp4 \
  --multi-voice --engine qwen

复现性能评测与 HTML

uv --cache-dir /private/tmp/voicebook-uv-cache run python \
  design/resources/qwen3ttsai/run_eval.py

uv --cache-dir /private/tmp/voicebook-uv-cache run python \
  design/resources/qwen3ttsai/build_html_report.py
